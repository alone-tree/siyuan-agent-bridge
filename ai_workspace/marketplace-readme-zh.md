# 思源桥

思源桥是一个面向 AI Agent 的本地 MCP 插件。它把你的思源笔记变成 Claude Code、Codex 等 AI 工具可以安全读取、搜索和维护的个人知识库。

## 它能做什么

- 在 AI 工具中列出、搜索和阅读思源笔记。
- 按块窗口阅读长文档，避免一次性塞入整篇笔记。
- 在需要编辑时使用块 ID 精确定位，减少误改。
- 支持创建文档、追加内容、替换块、编辑普通 Markdown 表格。
- 支持重命名、移动、复制、删除、导出文档。
- 通过隐私规则控制哪些笔记对 AI 隐藏、只读或可写。

## 安全设计

思源桥默认尊重本地和用户确认：

- 所有数据都在本机处理，不需要把整个笔记库上传到第三方服务。
- 隐私规则由你在思源中维护，AI 不能读取或编辑隐私规则文档。
- 写入类操作必须由用户明确要求，并且 MCP 调用参数中需要 `confirmed=true`。
- 写入思源前会创建工作空间快照，必要时可在思源中手动恢复。
- 只读文档可以被读取、复制或导出，但不能被编辑、移动、重命名或删除。

## 适合谁

适合已经在使用 Claude Code、Codex 或其他 MCP 客户端，并希望让 AI 更准确地使用自己思源笔记的人。

如果你只是想在思源内部聊天，本插件不是聊天机器人；它的重点是把思源作为外部 AI Agent 的知识库和写作环境。

## 使用方式

1. 安装并启用插件。
2. 打开插件设置页，确认 Python 命令和当前工作空间配置。
3. 复制插件生成的 MCP 配置。
4. 粘贴到 Claude Code、Codex 或其他 MCP 客户端中。
5. 在 AI 工具中让它先调用 `siyuan_start`，然后再搜索、阅读或维护笔记。

## 注意事项

- 需要本机已经安装 Python。
- 使用 MCP 时需要保持思源桌面端运行，并打开对应工作空间。
- 首次使用建议先在测试笔记本中验证读写流程。
- 删除、移动、覆盖等操作虽然有快照保护，但仍建议谨慎使用。
